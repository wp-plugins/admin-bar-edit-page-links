A simple plugin that adds a drop down list of pages to the admin bar.

== Description ==

Adds edit page links to the WordPress admin bar so you can quickly jump between editing pages. Very helpful if you're doing a lot of content editing.


== Installation ==

1. Upload the `admin-bar-edit-page-links` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
